## 🔧 TROUBLESHOOTING

### ❌ Deployment Failed on Render

**Problem:** Build fails with error message

**Solutions:**
1. **Check Render logs:**
   - Dashboard → Service → Logs
   - Look for specific error messages

2. **Common issues:**
   - Missing environment variable → Set in Render dashboard
   - Migration error → Check database connection
   - Package installation failed → Update requirements.txt
   - Port binding error → Render uses PORT env var automatically

3. **Force redeploy:**
   - Dashboard → Manual Deploy → Deploy Latest Commit

### ❌ Git Push Rejected

**Problem:** `! [rejected] main -> main (fetch first)`

**Solution:**
```bash
# Pull latest changes first
git pull origin main

# If conflicts, resolve them, then:
git add -A
git commit -m "Merge changes"
git push origin main
```

### ❌ Tests Failing in GitHub Actions

**Problem:** Green locally but red on GitHub

**Solution:**
1. Check GitHub Actions logs for specific error
2. Ensure all dependencies in requirements.txt
3. Check environment variables in workflow file
4. Run tests locally with same settings:
   ```bash
   SECRET_KEY=test DEBUG=True python manage.py test
   ```

### ❌ Render Not Auto-Deploying

**Problem:** Pushed to GitHub but Render didn't deploy

**Solution:**
1. Check `render.yaml` has `autoDeploy: true`
2. Verify webhook is configured:
   - Render Dashboard → Service → Settings
   - Should show GitHub connection
3. Manually trigger:
   - Dashboard → Manual Deploy → Deploy Latest Commit
